How to Register

To register Serial Logger Service, you must purchase it from www.muconsulting.com. After your payment is received and processed, you will be provided a registration code. Start the Serial Logger Service Configuration program, click Register from the Help menu, type in your registration name and code exactly as provided to you, and then click Validate Code. If the code you typed in is valid, close the registration window and restart the service. Viewing the Windows Event Log will confirm the registered version because it will not report an expiration time.
