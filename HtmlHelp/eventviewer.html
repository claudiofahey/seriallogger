Event Viewer

Event Viewer is a standard Windows application that displays events generated by Windows services and applications. It can be used to display events recorded by Serial Logger Service such as service start, stop, pause, continue, as well as timeouts, and configuration information. Serial Logger Service records such events in the Application Log. For more help with Event Viewer, click Start, then click Help, and then search the index for Event Viewer.

You can start Event Viewer by clicking Event Viewer from the Service menu. You can also use one of the following procedures.

If you are using Windows NT, you can open Event Viewer by clicking Start, then pointing to Programs, then pointing to Administrative Tools, and then clicking Event Viewer.

In Windows 2000, you can open Event Viewer by clicking Start, pointing to Settings, and then clicking Control Panel. Double-click Administrative Tools, and then double-click Event Viewer.
