Installation

Serial Logger Service can be installed on Windows NT 4.0, Windows 2000, and Windows XP. On Windows NT 4.0, you must have Microsoft Internet Explorer 4.0 or above. Because it uses advanced Windows features, Serial Logger Service does NOT run on Windows 95, Windows 98, or Windows Me.

To successfully install and configure Serial Logger Service, you must be logged in as an administrator.

To install Serial Logger Service, just run the single executable downloaded from the µ-Consulting web site at www.muconsulting.com.

The setup program will automatically perform the following steps:

  1. Install SerialLogger.exe (the actual service executable), SerialLoggerConfig.exe (the configuration executable), and SerialLogger.chm (compressed HTML Help file) to "C:\Program Files\Serial Logger Service."
  2. Create a Start Menu icon for the Serial Logger Service Configuration application.
