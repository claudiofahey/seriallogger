Revision History

Version 1.21e (5/5/2004)
  • Updated documentation.
Version 1.21d (8/16/2003)
  • Changed ODBC login timeout from 5 seconds to 120 seconds.
Version 1.21c (7/9/2003)
  • Modified the configuration application to work with Windows 2000 Service Pack 4.
Version 1.21a (8/23/2002)
  • Fixed bug that caused e-mailed log files to be truncated under certain conditions.
Version 1.21 (7/30/2002)
  • Added support for lines terminated with line feed + carriage return pairs (0A+0D).
  • When the serial port buffer is full, there is now a more useful warning message.
  • Fixed bug that caused some ODBC records to get lost when lines were received too fast.
  • Fixed bug that caused config application to give an access violation under certain environments.
Version 1.2 (6/24/2002)
  • No changes made.
Version 1.19a beta (6/17/2002)
  • Added ODBC database support.
  • Added button to e-mail the log file from the configuration application.
  • Help is now context-sensitive.
  • Added to documentation how to "login" to a serial device and how to rotate the log file on demand.
  • Fixed bug in timeout settings time control which will make it easier to use.
  • Various minor fixes to the configuration application.
Version 1.19 beta (5/14/2002)
  • Added support for multiple service instances.
  • Added menu option to launch Event Viewer.
Version 1.1 (4/29/2002)
  • No changes made.
Version 1.09b beta (4/15/2002)
  • Added capability to prefix each line in the log file with the current time.
  • Added capability to send e-mail to multiple recipients.
  • Fixed bug in SMTP e-mail module that caused very poor performance when e-mailing large attachments.
Version 1.09a beta (4/7/2002)
  • Added capability to e-mail log file after log rotation.
Version 1.09 beta (4/3/2002)
  • Added log rotation support.
  • There is no longer a limit to the amount of data that can be buffered while the service is paused.
  • Improved error handling and recovery when port read errors or log file write errors are encountered.
  • Fixed bug that forced one stop bit when parity was enabled.
  • Modified the core parts of the service to improve performance and flexibility.
  • Added minimize button to configuration application.
Version 1.0b (1/25/2002)
  • Added uninstall icon to start menu.
  • Minor changes to documentation to clarify certain items.
Version 1.0a (1/22/2002)
  • Added version number reporting to event log to easily identify the service version.
Version 1.0 (1/18/2002)
  • Initial release.
